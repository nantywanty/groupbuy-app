# Grouper App
Web application for coordinating and joining group purchases.

Developed by:
- Darius Yip

## React Front End
Requirements:
- Node version 19.7.0
- npm version 9.5.0

Navigate to frontend folder: `cd frontend`

Install required packages: `npm install`

Start the server: `npm start`
